Session Flows

How a learning session runs in each mode

💬 Tutor Mode
User enters topic
↓
Music auto-starts (lo-fi mood)
↓
AI streams intro chunk
↓ (repeats)
User asks question / continues
↓
AI responds in short chunk
↓ (user types /quiz)
Quiz generated + scored
↓
Progress saved → level updated
🤖 Agent Mode
User enters topic + duration
↓
Agent: get_progress → checks gaps
↓
Agent: set_music_mood (focus)
↓
Agent: search_web → fresh content
↓ (per subtopic)
Delivers chunk → progress bar advances
↓
Agent: generate_quiz
↓
Agent: mark_complete + save_note
↓
Progress saved → level updated

ADHD-Friendly UX Rules

Content chunks
Max 250 words per card. One idea at a time. Never a wall of text.
Always-visible progress
Progress bar + "Step 2 of 5" label visible at all times during session.
Minimal UI chrome
Focus mode hides everything except content + progress bar. One-click toggle.
Micro-rewards
Subtle animation + sound cue on completing each chunk and quiz. Level bar fills visibly.
No dead ends
Always show what comes next. "Next: Quiz" or "Next: Subtopic 2" — never blank state.
Music on by default
Music starts automatically on session begin (after first user gesture). Subtle volume fade-in.
