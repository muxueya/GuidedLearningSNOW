# Guided Learning

A smart, AI-powered learning tool that integrates music therapy to simplify learning, boost focus, improve memory retention, and enhance learning outcomes.

## Concept

Guided Learning pairs adaptive AI-generated content with scientifically-mapped background audio (binaural beats, lo-fi, ambient) to create an optimal cognitive environment for different types of study tasks.

| Task Type | Music Mode | Frequency |
|-----------|-----------|-----------|
| Deep focus | Gamma binaural beats | 40 Hz |
| Memorization | Alpha binaural beats | 10 Hz |
| Relaxed reading | Theta ambient | 6 Hz |

## Features

- **Adaptive Learning Sessions** — AI-generated explanations, summaries, and quizzes tailored to your level
- **Music Therapy Engine** — Automatic audio selection based on task type
- **Spaced Repetition** — Smart scheduling to reinforce what you've learned
- **Progress Tracking** — Session history, retention scores, and focus metrics

## Stack

- React + TypeScript (Vite)
- Tailwind CSS
- Tone.js / Howler.js (audio)
- Claude API (Anthropic) — adaptive content generation
- Zustand (state management)
